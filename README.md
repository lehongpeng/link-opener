# 链接批量处理工具

一个简单的网页工具，用于批量处理和打开链接列表。

## 功能特点

- 批量处理多个链接
- 自动检测链接可访问性
- 显示处理进度和时间
- 支持一键清空
- 错误处理和状态显示
- 苹果风格的界面设计

## 使用方法

1. 将链接列表粘贴到文本框中（每行一个链接）
2. 点击"处理链接"按钮
3. 等待处理完成
4. 查看处理结果

## 在线使用

## 本地使用

1. 下载 `link_opener.html` 文件
2. 使用浏览器打开该文件

## 注意事项

- 某些浏览器可能会阻止自动打开多个标签页
- 建议允许浏览器的弹出窗口权限
